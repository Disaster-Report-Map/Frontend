DRM Frontend scaffold

Setup:

1. Copy `.env.example` to `.env.local` and fill values.
2. Install dependencies:

```bash
npm install
npm install axios zustand socket.io-client react-hook-form zod @hookform/resolvers mapbox-gl clsx
npm install -D @types/mapbox-gl tailwindcss postcss autoprefixer typescript
```

3. Run dev server:

```bash
npm run dev
```
